# React Web Example - @laddhaanshul/layout-spacing

This example demonstrates how to use `@laddhaanshul/layout-spacing` in a React web application built with Vite.

## Features Demonstrated
- **LayoutSpacingProvider**: Global configuration for the 8pt grid.
- **Layout Primitives**: Extensive use of `Box`, `Flex`, and `Stack`.
- **Responsive Hooks**: Breakpoint detection and responsive values.
- **Aspect Ratio**: The `AspectRatio` component for images and containers.

## Running Locally

1. Install dependencies from the monorepo root:
   ```bash
   pnpm install
   ```

2. Start the development server:
   ```bash
   pnpm example:web
   ```

3. Open your browser at `http://localhost:5173`.

## Key Files
- `src/App.tsx`: Main demonstration page.
- `src/main.tsx`: Provider setup.
