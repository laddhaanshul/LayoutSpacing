## Contributing

We welcome contributions! Please follow these guidelines:

### Getting Started
1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/layout-spacing.git`
3. Install dependencies: `pnpm install`
4. Create a feature branch: `git checkout -b feature/your-feature`
5. Make your changes
6. Run tests: `pnpm test`
7. Commit your changes: `git commit -m 'Add your feature'`
8. Push to your fork: `git push origin feature/your-feature`
9. Open a Pull Request

### Code Style
- TypeScript strict mode
- Use the existing code patterns as reference
- Components use `React.createElement` (no JSX) for cross-platform compatibility
- All exports must be properly typed

### Commit Convention
- `feat:` New features
- `fix:` Bug fixes
- `docs:` Documentation changes
- `test:` Test additions or changes
- `refactor:` Code refactoring
- `chore:` Build/CI changes

### Pull Request Checklist
- [ ] All tests pass
- [ ] New code has test coverage
- [ ] Documentation is updated
- [ ] No TypeScript errors
- [ ] Changes follow existing patterns
